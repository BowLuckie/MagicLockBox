# **Magic Lock Box**

To get started with the **Magic Lock Box**, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/BowLuckie/MagicLockBox
   ```

2. **Navigate to the project directory:**

   ```bash
   cd MagicLockBox
   ```

3. **Run the application:**

   ```bash
   python -m "magic lock"
   ```
